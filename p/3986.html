苹果最新 macOS Monterey 蒙特雷 12.0 操作系统

苹果 macOS Monterey (蒙特雷) 在 UI 界面、图标和交互上基本沿用了上一版本的设计，这一次更加注重于“生产力”的加强，得益于新 Mac 的 M1 处理器加持，Monterey 不仅可以让你在 Mac 桌面端运行 iOS 应用，一骑绝尘的性能也能让你体验到丝般顺滑的流畅。

「下载 macOS Monterey + iOS 15 全套内置壁纸」

继跨设备的“通用剪贴板”、“应用接力”、“隔空投送”之后，苹果生态再次带来了杀手级的功能——Universal Control (通用控制 / 键鼠共享)。你可以使用当前 MacBook 的键盘鼠标/触控板，来“隔空操作”你旁边的 iPad 或 iMac，并能在设备之间拖放文件图片。这一点让苹果设备之间不再有隔阂，真心能大大提高日常操作的效率。

同时苹果也在 macOS Monterey 上引入了「AirPlay to Mac」。该功能可以让用户将 Mac 变成一台投屏接收设备。你可以将 iPhone、iPad 上的视频或音频，通过 AirPlay “隔空播放”投屏到 Mac 的屏幕/音箱上播放，也支持屏幕镜像。

另外，还有捷径 for Mac、实时文本 (OCR 识别图片照片文字)，隐私保护加强、新版 Safari 浏览器、SharePlay、快捷备忘录、专注模式等一系列新功能加入。

macOS Monterey (WWDC) 介绍视频：

各种新特性使得 macOS Big Sur 不仅功能更加的丰富好用，而且稳定性和兼容性也依然会有保障。下面我们一起来看看苹果最新的 macOS 12.0 系统都带来了哪些重点更新的功能特性值得我们升级的吧。

Universal Control 苹果的“多屏协同”：

这是 macOS Monterey 的重磅新功能，名为 Universal Control“全局控制”。它能让用户只用一套键盘、鼠标或触控板，用户就能在 Mac 和 iPad 之间“隔空”操控，比如 MacBook 的鼠标指针可以跨越屏幕跑到 iPad 上或 iMac 上去，并且可以从 A 拖拽文件放到 B 或 C 去使用，享受完全无缝的跨屏协作体验。

这一功能类似罗技旗舰鼠标主打的 Flow 工具或 Synergy 等第三方键鼠工具，不过组建多屏连接时更加简单，根据演示似乎只需将机器放在旁边即可自动实现“鼠标穿透”。这让 Mac 和 iPad之间有了更强大的互通协作能力，对于经常要在不同设备上创作内容的人来说将会非常有用。

AirPlay to Mac

让你的 Mac 支持「接收 AirPlay 隔空播放」的功能，你可以将 iPhone、iPad 的屏幕内容镜像投屏到 MacBook 或 iMac 上显示，也能将视频、音乐等媒体隔空播放到 Mac 上去，让你更加有效地利用 Mac 显示效果出色的大屏幕以及高清音响。

Mac 平台的「快捷指令」

iOS 上的「快捷指令」功能可以让很多不太熟悉编程代码的朋友也享受到“自动化”流程带来的效率提升，而现在你终于也可以在 macOS 上使用各类方便的“捷径”了。

你可以实现一些比如快速打开资料库、共享选定的文件、编辑器，或比如快速打开 Pinterest、Affinity Photo 应用，同时打开 Apple Music 等等，让从事大量创意工作的用户，生产力效率大幅提升。现在，你可以让日常工作高效自动化，一切都变的更加轻松快捷。

Safari 浏览器 大量改进

在 Chrome 和 Edge 的强攻之下，苹果也对 Safari 浏览器进行了大量的改进，全新的 UI 界面将标签栏和地址栏合二为一，显得更加紧凑、现代和清新。多个打开的标签页可以快速保存分组，并且分组可以在不同设备上自动同步。

并且针对“隐私保护”做了大量的工作，优化了续航，更节能，新增了一个低功耗模式（Low Power Mode）的模式，并且现在 iPadOS 和 iOS 上的 Safari 终于也都支持浏览器扩展了！

更多功能：

还有一些列的小改进就不再详细介绍了，大家可以下载回去自行体验发现。而针对开发者，苹果还推出了 XCode Cloud 与 TestFlight for Mac 等功能。

当然，macOS Monterey 也继承了之前版本比如 macOS Big Sur、Catalina、Mojave 的全部实用功能特性。比如 Sidecar (将 iPad 用作 Mac 的扩展显示器并支持触摸操作与 Apple Pencil 手写)、通用剪贴板、应用接力、隔空投送、手机摄像头接力等，在 Monterey 都能继续享用。

macOS Monterey 官方 dmg 镜像下载：

和以往一样，苹果官方并不提供 macOS Monterey 的 dmg 格式镜像文件。如需进行全新的抹盘安装，你只能下载 macOS 的升级安装程序后，自己制作 USB 启动盘使用。

总结：

随着 macOS 的不断升级以及与 iPhone、iPad 的互动功能越来越多，比如跨屏共享键鼠、跨设备拖放文件、将 iPad 当做 Mac 的显示器 、跨设备复制粘贴、AirPlay 投屏等，甚至现在你还能在 Mac 上运行丰富的 iOS 应用，整个苹果生态真的相当牛逼。

可以说，Mac + iPhone 能带给你 Windows + 安卓 完全不能比拟的优势。所以如果你准备换电脑的话，那么真的不妨考虑下 iPhone + Mac 系列吧。
